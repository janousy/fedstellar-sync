class fedstellar.learning.pytorch.lightninglearner.LightningLearner(model, data, logger=None)#

Bases: NodeLearner

Learner with PyTorch Lightning.

Atributes:

model: Model to train. data: Data to train the model. epochs: Number of epochs to train. logger: Logger.
