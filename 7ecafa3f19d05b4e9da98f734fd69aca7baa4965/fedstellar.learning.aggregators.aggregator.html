fedstellar.learning.aggregators.aggregator module#

class fedstellar.learning.aggregators.aggregator.Aggregator(node_name='unknown', config=None)[source]#

Bases: Thread, Observable

Class to manage the aggregation of models. It is a thread so, aggregation will be done in background if all models were added or timeouts have gone. Also, it is an observable so, it will notify the node when the aggregation was done.

Parameters:

node_name – (str): String with the name of the node.

add_model(model, nodes, weight)[source]#

Add a model. The first model to be added starts the run method (timeout).

Parameters:

  • model – Model to add.

  • nodes – Nodes that collaborated to get the model.

  • weight – Number of samples used to get the model.

check_and_run_aggregation(force=False)[source]#

Check if all models have been added and start aggregation if so.

Parameters:
