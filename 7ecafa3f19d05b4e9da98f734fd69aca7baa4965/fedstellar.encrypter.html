Class with methods to encrypt and decrypt messages using AES symetric encryption.

add_padding(msg)[source]#

Add padding to an encoded UTF-8 text. Adds “ “ charactets (1 byte) to fill the rest with the block. Careful: in this case, the filling content doesn’t affect to the meaning of messages.

Parameters:

msg – (bytes) The encoded text.

Returns:

(bytes) The encoded text with padding.

Return type:

msg

decrypt(message)[source]#

Decrypts a message using AES. Message is decripted using the shared key. Keep in mind that AES uses a block cipher, so the message can be a filled message with padding.

Parameters:

message – (bytes) The message to decrypt.

Returns:

(bytes) The decrypted message.

Return type:

message
